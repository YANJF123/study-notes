### basic linux operation 基础操作

#### 学会使用命令帮助
whatis info man which whereis
```linux

// 查看某个命令的简要说明
whatis commond

// 正则匹配,搜索含有local字符的命令
whatis -w "local*"

// 查看命令的详细说明
info commond

// 查看命令的说明文当
man commond

// 用来搜索只记住了部分关键字的命令
man -k commond

// 查看命令程序的binary安装路径
which commond

// 查看程序的搜索路径
whereis commond
```
