# 1 php

### 基础

+ 标记
  + 有多种标记,推荐的标记依然是XML风格`<?php ?>`
+ 空格
  + 换行(回车),空格和Tab(制表符)都被认为是空格.
+ 注释
  + php支持C,C++和SHELL脚本风格的注释
  + /* here is comment */
  + // here is comment
  + # here is comment
+ 访问表单变量
  + 有多重方法获取表单数据,其中**中等风格**是如今推荐的.
  + `$_POST['name']`
+ 字符串的连接
  +  用`.`
  + `echo $name.' is good!'`
+ 变量字符串
  + 用双引号和单引号括起来的字符串php解析器的处理会不同
  + 用双引号括起来的字符串php解析器会尝试解析其中的变量
  + 用单引号括起来的字符串会被当做真正的文本.
  + 可以用heredoc语法指定字符串
  
  ```php
   echo <<<HTML
   line 1
   line 2
   HTML
  ```
